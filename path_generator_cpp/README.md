# 路径生成器 (Path Generator)

基于 C++11 的线性插值路径生成器，沿矩形边缘生成最短路径，支持梯形速度规划、yaw 旋转标志插入、红蓝区镜像等功能。

## 快速开始

```bash
# 编译为库（不含 main）
g++ -std=c++11 -DPATH_GENERATOR_NO_MAIN -c path_generator.cpp -lm
```

在你的代码中调用：

```cpp
#include "path_generator.h"

int main() {
    // 1. 配置速度参数
    PathConfig config;
    config.max_vel = 2.0f;   // 最大速度 (m/s)
    config.accel   = 2.0f;   // 最大加速度 (m/s²)
    config.decel   = 1.5f;   // 最大减速度 (m/s²)

    // 2. 生成路径（中间点编号1和7，红区）
    ThreeSegmentResult result = generate_path_from_waypoint_numbers(1, 7, config, 1);

    // 3. 使用结果
    // result.seg1 / result.seg2 / result.seg3 为三段路径 (std::vector<PathPoint>)
    // result.selected_wp1_num / result.selected_wp2_num 为排序后的中间点编号

    // 4. 也可以直接访问全局变量（调用后自动填充）
    // extern std::vector<PathPoint> g_seg1;  // 第一段路径
    // extern std::vector<PathPoint> g_seg2;  // 第二段路径
    // extern std::vector<PathPoint> g_seg3;  // 第三段路径
    // 每个元素为 PathPoint {vx, vy, x, y, yaw}，单位: mm/s, mm/s, mm, mm, rad

    // 5. 保存到文件（不需要可删除）
    save_three_segments(result.seg1, result.seg2, result.seg3,
                        "my_path.txt");

    return 0;
}
```

有效编号为 **1, 2, 3, 4, 6, 7, 9, 10, 11, 12**（编号 5 和 8 无效）。

## 可配置参数

所有参数均在 `path_generator.h` 中修改，修改后需重新编译。

### 地图边界

| 参数 | 默认值 | 说明 |
|:-----|:------:|:-----|
| `RECT_X_MIN` | 2600.0 | 下边界 x (mm) |
| `RECT_X_MAX` | 8600.0 | 上边界 x (mm) |
| `RECT_Y_MIN` | 600.0  | 右边界 y (mm) |
| `RECT_Y_MAX` | 5400.0 | 左边界 y (mm) |

### 起点与终点

| 参数 | 默认值 | 说明 |
|:-----|:------:|:-----|
| `FIXED_START_X`   | 2600.0 | 总起点 x |
| `FIXED_START_Y`   | 5400.0 | 总起点 y |
| `FIXED_START_YAW` | 0.000  | 总起点 yaw (rad) |
| `FIXED_END_X`     | 8600.0 | 总终点 x |
| `FIXED_END_Y`     | 600.0  | 总终点 y |
| `FIXED_END_YAW`   | 0.000  | 总终点 yaw (rad) |

### 路径与速度

| 参数 | 默认值 | 说明 |
|:-----|:------:|:-----|
| `PATH_POINT_INTERVAL` | 150.0 | 插值点间隔 (mm) |
| `DEFAULT_MAX_VEL`     | 2.0   | 最大速度 (m/s) |
| `DEFAULT_ACCEL`       | 2.0   | 最大加速度 (m/s²) |
| `DEFAULT_DECEL`       | 1.5   | 最大减速度 (m/s²) |

### 偏移与区域

| 参数 | 默认值 | 说明 |
|:-----|:------:|:-----|
| `WAYPOINT_OFFSET_DISTANCE` | 298.940 | 中间点偏移距离 (mm) |
| `ZONE` | -1 | 区域选择：1=红区，-1=蓝区，0=不输出 |

> **偏移规则**：WP1 的投影点和目标点沿车右方向偏移，WP2 沿车左方向偏移。

## 输出格式

输出文件包含三个 C 数组，每行 5 列：

```
Vx(mm/s),  Vy(mm/s),  X(mm),  Y(mm),  yaw(rad)
```

### 输出示例

```c
const float path_segment_1[][5] = {
{      0.000f,      0.000f,    2600.000f,    5400.000f,    0.000f},  // 起点: Vx=Vy=0
{      0.000f,    774.597f,    2600.000f,    5250.000f,    0.000f},  // 正常行驶点
{      0.001f,      0.001f,    2600.000f,     600.000f,   -1.571f},  // ⬅ 旋转标志行
{     10.000f,      0.000f,    2600.000f,     600.000f,   -1.571f},  // 角落点（低速）
{      0.000f,      0.000f,    2750.000f,     650.000f,    1.571f},  // 目标点: Vx=Vy=0（停车）
};

uint16_t path_segment_sizes[3] = {28, 34, 21};
```

### 特殊行识别

| 行类型 | Vx | Vy | 说明 |
|:------:|:--:|:--:|:-----|
| 起点/终点 | 0 | 0 | 每段首尾 |
| 目标点 | 0 | 0 | 中间停车点 |
| 旋转标志 | 0.001 | 0.001 | 车体原地旋转，yaw 发生跳变 |
| 角落点 | 低速 | 低速 | 矩形拐角处，速度 ≤ 10 mm/s |

## 路径结构说明

输入两个中间点编号后，生成器会自动选择最短路径顺序，将路径分为三段：

```
总起点 ──[沿边缘]──▶ WP1投影点 ──[垂直]──▶ WP1目标点（停车）
                                                     │
                  Seg1                               │ Seg2
                                                     ▼
总终点 ◀──[沿边缘]── WP2投影点 ◀──[垂直]── WP2目标点（停车）
                                                     │
                  Seg3                               │
                                                     ▼
                                              WP2目标点（停车）
```

- **Seg1**：总起点 → WP1投影点（沿边缘）→ WP1目标点（垂直前往，停车）
- **Seg2**：WP1目标点 → WP1投影点（垂直返回）→ WP2投影点（沿边缘）→ WP2目标点（垂直前往，停车）
- **Seg3**：WP2目标点 → WP2投影点（垂直返回）→ 总终点（沿边缘）

## 红蓝区配置

通过 `ZONE` 参数或 `generate_path_from_waypoint_numbers` 的 `zone` 参数设置：

| 值 | 区域 | 处理 |
|:--:|:----:|:-----|
| 1 | 红区 | 原始坐标直接输出 |
| -1 | 蓝区 | y→-y, vy→-vy, yaw→-yaw（旋转标志和首尾点的 vy 不取反） |
| 0 | 无 | 不输出文件 |

## 编译选项

```bash
# 编译为目标文件（作为库使用，推荐）
g++ -std=c++11 -DPATH_GENERATOR_NO_MAIN -c path_generator.cpp -lm

# 编译为独立可执行文件（仅用于调试）
g++ -std=c++11 -o path_generator path_generator.cpp -lm
```

> **注意**：需要 C++11 标准，链接数学库 `-lm`。使用 `-DPATH_GENERATOR_NO_MAIN` 宏可禁用内置 main 函数，避免与你的程序入口冲突。
